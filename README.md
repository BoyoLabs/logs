# logs
Logs for my exploits.

I do a lot of AI augmented work. Instead of continuing with RAG, I am shifting more to a log based system that will help streamline future repetitive tasks, and help serve as a provenance for the work to get to a certain product. This repository keeps those.

## provenance vs RAG

There will be 2 different folders in here, both are effectively logs. However, some logs serve as a RAG in the sense that its provenance will serve the AI in the future to save tokens and time.
